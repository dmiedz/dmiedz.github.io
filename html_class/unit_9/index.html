Danny's Dougnuts was started in 2018, when Danny Miedzinski signed up for an HTML, CSS, and Web Design circuit to better understand some of the web products that he works on. Danny decided to create a fake doughnut restaurant, and build the website for it. He's still not entirely sure if it is spelled "doughnut" or "donut".
